# BoardCost

Instant cost estimator for point-of-sale cardboard display units (FSDUs,
CDUs, standees and dump bins). A browser-based quoting tool built as a
degree apprenticeship software engineering project.

> Full project report (design, planning, build narrative, TDD, CI/CD and
> evaluation) will live in this README as the project progresses.
